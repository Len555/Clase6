# explicacion
# Ejercicio 1 el sistema de control de temperatura se implemento utilizando programa modular, dividiendo el problema en funciones especificas: leer_sensores, calcular_temperatura_optima; considerando el dia la ocupacion y el clima.
# EJERCICIO 2 EL SISTEMAse implemento utilizando programacion modular mediante funciones que separan las distintas responsabilidades del sistema: registrar_entrada, registrar_salida, calcular_inventario_optimo. se determina el nivel adecuado basado en la demanda del promedio del producto: genera_alerta_restablecimiento.
# Ejercicio 3 el sistema de navegacio se implemetno utilizando programacion modular mediante funciones que separan las tareas del vehiculo autonomo. leer_sensores: obtiene los datos de sensores de proximidad que permite detaectar la distancia de obstaculos cercanos. calcular_ruta_optima: determina una ruta entre el origen y el destino. detectar_obstaculo: evalua si existe un obstaculo cercano utilizando la informacion de los sensores. ajuste_velocidad: controla la velocidad del vehiculo.
